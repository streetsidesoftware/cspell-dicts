# dictionary-ro

Romanian (or Moldavian; or Moldovan) spelling dictionary.

## What is this?

This is a Romanian (or Moldavian; or Moldovan) dictionary,
generated by [`wooorm/dictionaries`][dictionaries] from
[`rospell.wordpress.com`][source],
normalized and packaged so that it can be installed and used like other
dictionaries.

## When should I use this?

You can use this package when integrating with tools that perform spell checking
(such as [`nodehun`][nodehun], [`nspell`][nspell]) or when making such tools.

## Install

In Node.js (version 12.20+, 14.14+, or 16.0+), install with [npm][]:

```sh
npm install dictionary-ro
```

## Use

```js
import dictionaryRo from 'dictionary-ro'

dictionaryRo(function (error, ro) {
  if (error) throw error
  console.log(ro)
  // To do: use `ro` somehow
})
```

Yields:

```js
{dic: <Buffer>, aff: <Buffer>}
```

Where `dic` and `aff` are [`Buffer`][buffer]s for `index.dic` and `index.aff`
respectively.
